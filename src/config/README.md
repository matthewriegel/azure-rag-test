# Configuration

Environment-based configuration management with validation, feature flags, and structured config objects for all services.

## Config Structure

```typescript
interface Config {
  env: 'development' | 'production' | 'test';
  port: number;
  azureOpenAI: { endpoint, apiKey, chatDeployment, embeddingDeployment };
  azureSearch: { endpoint, apiKey, indexName };
  azureRedis: { host, port, password, tls };
  azureStorage: { accountName, accountKey, containerName };
  rag: { chunkSize, chunkOverlap, searchTopK, confidenceThreshold };
  api: { ingestApiKey, rateLimitWindow, rateLimitMax };
}
```

const featureFlags = {
