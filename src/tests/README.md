# Tests

Jest-based unit tests covering confidence calculation, tokenization, helpers, and key service logic.

## Test Structure

```
tests/
├── setup.ts           # Jest configuration
├── confidence.test.ts # Confidence scoring tests
├── tokenizer.test.ts  # Token counting & chunking tests
└── helpers.test.ts    # Utility function tests
```

## Running Tests

```bash
# Run all tests
npm test

# Watch mode
npm run test:watch

# Coverage report
npm run test:coverage
```

## Example Test

```typescript
import { calculateConfidence } from '../services/confidence';

describe('Confidence', () => {
  it('calculates weighted score', () => {
    const result = calculateConfidence({
      avgSimilarityScore: 0.9,
      avgLexicalScore: 40,
      llmScore: 0.85
    });
    expect(result.finalConfidence).toBeCloseTo(0.72, 2);
  });
});
```
