# Scripts

Automation scripts for local setup and Azure deployment.

## Available Scripts

### `setup-local.sh`
Sets up local development environment:
- Starts Docker Compose (Redis)
- Copies `.env.local.example` if needed
- Installs npm dependencies
- Runs initial build

```bash
bash scripts/setup-local.sh
```

### `deploy-azure.sh`
Deploys infrastructure and application to Azure:
- Creates resource group
- Deploys Bicep templates
- Builds and pushes Docker image to ACR
- Updates Container App

```bash
bash scripts/deploy-azure.sh
```
